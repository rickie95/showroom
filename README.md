# Showroom

## Overview

**Showroom** is frontend for [Garage](https://garagehq.deuxfleurs.fr/), an S3-compatible object storage service. Garage is great, but lacks an UI. Withk Showroom you can:

- check your cluster's health
- manage your buckets and your keys

![](./assets/dashboard.png)

> Showroom is currently compatible only with V1 API. V2 API support is coming soon.
